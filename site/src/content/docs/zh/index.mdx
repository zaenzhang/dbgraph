## 快速路径

```bash
curl -fsSL https://raw.githubusercontent.com/zaenzhang/dbgraph/master/install.sh | sh
cd your-project
dbgraph init -i --yes
dbgraph doctor
dbgraph snapshot --profile stats
dbgraph analyze --scope all --format markdown --output dbgraph-analysis.md
```
